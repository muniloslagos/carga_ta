REFERENCIA DE TIPOS:
────────────────────

i = INTEGER
  └─ Para valores numéricos enteros

  └─ Para valores numéricos decimales

b = BLOB
  └─ Para datos binarios

                    REGLA DE CONCORDANCIA EN BIND_PARAM()
═══════════════════════════════════════════════════════════════════════════════

REGLA:
──────
Número de caracteres en el string = Número de variables
