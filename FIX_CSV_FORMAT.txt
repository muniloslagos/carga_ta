                      LISTA COMPLETA DE TIPOS PERMITIDOS
═══════════════════════════════════════════════════════════════════════════════

✅ DOCUMENTOS

• PDF (.pdf)
  Archivos de documento portátil
  Uso: Reportes, formularios, documentación
  
• DOC / DOCX (.doc, .docx)
  Documentos de Microsoft Word
  Uso: Informes, comunicados, actas
  
• XLS / XLSX (.xls, .xlsx)
  Hojas de cálculo Excel
  Uso: Datos, estadísticas, registros

• CSV (.csv) ← NUEVO
  Archivos de valores separados por comas
  Uso: Datos tabulares, importación/exportación

✅ IMÁGENES

• JPG / JPEG (.jpg, .jpeg)
  Imágenes comprimidas
  Uso: Fotografías, capturas de pantalla
  
• PNG (.png)
  Imágenes sin pérdida de calidad
  Uso: Gráficos, diagramas, imágenes con transparencia

└─ descripcion: "Archivo de datos"

PASO 3: Servidor valida la extensión
──────────────────────────────────────

$fileExt = strtolower(pathinfo($file['name'], PATHINFO_EXTENSION));
// $fileExt = 'csv'

if (!in_array($fileExt, $allowedTypes)) {
    // ANTES: true (CSV no estaba en la lista) → ERROR
    // AHORA: false (CSV está en la lista) → PERMITIDO
    return ['error' => 'Tipo de archivo no permitido'];
}

PASO 4: Se valida el tamaño
─────────────────────────────

if ($file['size'] > $maxSize) {
    return ['error' => 'El archivo es demasiado grande (máximo 10MB)'];
}

PASO 5: Se sube el archivo
──────────────────────────

move_uploaded_file($file['tmp_name'], $filepath);
// Archivo guardado en: uploads/doc_[timestamp].csv

PASO 6: Se registra en la base de datos
────────────────────────────────────────
