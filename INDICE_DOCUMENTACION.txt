═══════════════════════════════════════════════════════════════════════════════
                        ARCHIVOS DE CONFIGURACIÓN
═══════════════════════════════════════════════════════════════════════════════

🔧 check.php
   Validador de requisitos del sistema
   - Verifica versión de PHP
   - Verifica extensiones necesarias
   - Comprueba permisos de archivos
   - Valida conectividad a BD
   → Acceso: http://localhost/cumplimiento/check.php
   → Usar: Antes de setup.php

🔧 setup.php
   Script de instalación y creación de BD
   - Crea base de datos
   - Crea todas las tablas
   - Inserta usuario administrativo
   - Crea índices
   → Acceso: http://localhost/cumplimiento/setup.php
   → Usar: Después de check.php (primera vez)

🔧 ejemplo_datos.php
   Script para insertar datos de ejemplo
   - Crea 5 direcciones
   - Crea 4 usuarios de prueba
   - Crea 15 items de transparencia
   - Asigna usuarios a items
   → Acceso: http://localhost/cumplimiento/ejemplo_datos.php
   → Usar: Opcional, para pruebas

═══════════════════════════════════════════════════════════════════════════════
                        ARCHIVOS DE CÓDIGO
═══════════════════════════════════════════════════════════════════════════════

📂 config/
   └─ config.php              Configuración global del sistema
      - Credenciales de BD
      - URL del sitio
      - Constantes de perfil
      - Configuración de periodicidades
   
   └─ Database.php            Clase para conexión a BD
      - Métodos de conexión
      - Métodos de query
      - Métodos de transacciones

📂 classes/
   └─ Usuario.php             Modelo de Usuario
      - CRUD de usuarios
      - Autenticación
      - Cambio de contraseña
   
   └─ Direccion.php           Modelo de Dirección
      - CRUD de direcciones
      - Gestión de usuarios por dirección
   
   └─ Item.php                Modelo de Item de Transparencia
      - CRUD de items
      - Asignación de usuarios
      - Obtención de items por usuario
   
   └─ Documento.php           Modelo de Documento
      - CRUD de documentos
      - Validación de archivos
      - Gestión de estado

📂 admin/                     Panel de Administración
   └─ index.php               Dashboard administrativo
   
   └─ usuarios/
      ├─ index.php            Gestión de usuarios
      └─ get_usuario.php      API JSON de usuario
   
   └─ direcciones/
      ├─ index.php            Gestión de direcciones
      └─ get_direccion.php    API JSON de dirección
   
   └─ items/
      ├─ index.php            Gestión de items
      ├─ get_item.php         API JSON de item
      └─ get_usuarios_item.php API JSON de usuarios asignados
   
   └─ documentos/
      └─ index.php            Revisión de documentos

📂 usuario/                   Panel de Usuario
   ├─ dashboard.php           Panel principal
   └─ enviar_documento.php    Procesamiento de carga de documentos

📂 includes/
   ├─ header.php              Encabezado común (navbar)
   └─ footer.php              Pie de página
